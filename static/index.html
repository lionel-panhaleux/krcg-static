VTES Static Files

This is a server hosting development resources for V:tES (Vampire: the Eternal Struggle) developers.

This server is optimised for performance: files are served under a 1 hour Cache-Control.

Data resources

Data resources are under the data directory.

  • data/vtes.json contains all cards data with translations, rulings and image URL, in JSON format.
  • data/twda.json contains a list of the TWDA, in JSON format.
  • data/twd.htm is a normalized version of the official TWDA. It's far easier to parse, as it uses a homogenous format in all places, and all card names are correct.

Web resources

The example.html provides a short documentation and an actual example of their use.

Web resources are under the web directory.

  • web/ankha.ttf is a font used to display disciplines and card types.
  • web/krcg.js is a simple script to display a card image modal.
  • web/krcg.css is a simple CSS for easy usage of the previous items.

Card Images

Card images are JPG files under the card directory. Images filenames are an ascii-only version of card names, with neither space nor punctuation, and the .jpg file extentions. For example: card/powerbasezurich.jpg. These images URLs are used in the vtes.json file.

Clan images

Card images are SVG files under the clan directory. They display nicely at any size. Images filenames are an ascii-only version of clan names, with neither space nor punctuation, and the .svg file extentions. For example: clan/daughtersofcacophony.svg.

V5 clan icons are available under the clan/v5 folder. For example: clan/v5/tremere.svg.
