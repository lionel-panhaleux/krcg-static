VTES Static Files

This is a server hosting development resources for V:tES (Vampire: the Eternal Struggle) developers. It is optimised for performance: files are served under a 1 hour Cache-Control.

The files available here are shared under the conditions of the agreement.
They are free to use under the same conditions.

Data resources

Data resources are under the data directory.

  • https://static.krcg.org/data/vtes.json all cards data, with rulings and translations, in JSON format.
  • https://static.krcg.org/data/twda.json the TWDA in JSON format.
  • https://static.krcg.org/data/twd.htm a normalized version of the official TWDA.
  • https://static.krcg.org/data/countries.json a JSON version of geonames.org countries list.
  • https://static.krcg.org/data/cities.json a JSON version of geonames.org major cities list.

Web resources

The example.html provides a short documentation and an actual example of their use.

Web resources are under the web directory.

  • https://static.krcg.org/web/ankha.ttf a font to display disciplines and card types.
  • https://static.krcg.org/web/vtes-clans.otf a font to display clans icons.
  • https://static.krcg.org/web/krcg.js a script to display a card image modal.
  • https://static.krcg.org/web/krcg.css a basic CSS to enable the previous items.

Card Images

Card images are JPG files under the card directory. Images filenames are an ascii-only version of card names, with neither space nor punctuation, and the .jpg file extentions.
For example: https://static.krcg.org/card/powerbasezurich.jpg

The translated images, when available, can be found in the subdirectory matching the language, under the same filename as the original card image.
For example: https://static.krcg.org/card/fr/fame.jpg

These images URLs are used in the vtes.json file.

Clan images

Card images are SVG files under the clan directory. They display nicely at any size. Images filenames are an ascii-only version of clan names, with neither space nor punctuation, and the .svg file extentions.
For example: https://static.krcg.org/clan/daughtersofcacophony.svg

V5 clan icons are available under the clan/v5 folder.
For example: https://static.krcg.org/clan/v5/tremere.svg
